  <files>
    web/app/api/settings/api-key/route.ts
  </files>
  <action>
Create `web/app/api/settings/api-key/route.ts` with two handlers:

**POST handler** (store API key):
1. Get session via `getSession()`. Return 401 if no `session.userId`.
2. Parse JSON body: `{ apiKey }`. Return 400 if missing or empty.
3. Trim the API key. Validate format: must start with `sk-ant-` (Anthropic key prefix). Return 400 with "Invalid API key format" if wrong.
4. Call `validateApiKey(apiKey)` from `@/lib/anthropic` to test the key against the Anthropic API.
5. If invalid, return 400 with the error message from `validateApiKey`.
6. If valid, encrypt the key: `const encrypted = encrypt(apiKey)` from `@/lib/crypto`.
7. Write encrypted key to `{USERS_DIR}/{session.userId}/api_key.enc` using `writeFile` from `node:fs/promises`.
8. Return `Response.json({ ok: true })`.

**GET handler** (check if key exists):
1. Get session. Return 401 if no userId.
2. Check if file exists: `{USERS_DIR}/{session.userId}/api_key.enc` using `fs.access()` or try/catch on `readFile`.
3. Return `Response.json({ hasKey: true/false })`.

Import `USERS_DIR` from `@/lib/constants`. Use `path.join(USERS_DIR, session.userId, 'api_key.enc')` for the file path.
  </action>
